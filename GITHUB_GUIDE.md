# GitHub Pages 업로드 가이드 (5분)

## 1단계: GitHub 접속
https://github.com → 로그인 or 회원가입

## 2단계: 새 저장소 만들기
- 우측 상단 + → New repository
- Repository name: energy-quest
- Public 선택 (중요!)
- Create repository 클릭

## 3단계: 파일 업로드
- "uploading an existing file" 클릭
- 이 폴더 안의 파일 전체 드래그 앤 드롭
  (GITHUB_GUIDE.md 제외하고 나머지 전부)
- Commit changes 클릭

## 4단계: GitHub Pages 활성화
- Settings 탭 → Pages (좌측 메뉴)
- Source: Deploy from a branch
- Branch: main → /(root) → Save

## 5단계: URL 확인 (1~2분 후)
https://[내아이디].github.io/energy-quest

이 URL을 카톡으로 공유하면 끝!

## 학생들 앱처럼 설치 방법
1. 크롬에서 URL 열기
2. 주소창 옆 메뉴(⋮) → "홈 화면에 추가"
3. 아이콘 생성됨 → 앱처럼 전체화면 실행
