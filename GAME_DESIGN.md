# 에너지 퀘스트 — 게임 설계 문서 (에너지 여정 v1 초안)

> 목적: 흩어진 스테이지 주제를 **"에너지"라는 하나의 축**으로 재정렬하기 위한 기준 문서.
> 엔진·아트·게임플레이는 그대로 두고, **3~5단계의 내용(퀴즈·대사·아이템·연출)만 에너지로 교체**한다.

---

## 1. 한 줄 컨셉 (로그라인)
> DEC(다크에너지 코퍼레이션)가 에너지를 독점·차단한 세상에서, 어린 공학자가 **신재생에너지와 에너지 하베스팅**으로 마을의 발전을 하나씩 되살려 **에너지 자립**을 이루고 DEC에 맞선다.

## 2. 핵심 메시지 (왜 배우는가 — 학습 목표)
1. **에너지 안보**: 에너지를 남이 쥐면 삶 전체가 위태롭다. (정전 → 냉장고·병원·가로등 멈춤)
2. **신재생에너지**: 태양·바람 등 고갈되지 않는 에너지로 스스로 만들 수 있다.
3. **에너지 하베스팅**: 버려지는 자투리 에너지(진동·열·압력)도 모으면 자원이 된다.
4. **저장·스마트그리드**: 만든 에너지를 저장하고 똑똑하게 나눠야 진짜 **자립**이 완성된다.
5. **결론**: *우리 손으로 만들고 관리하는 에너지 = 누구에게도 휘둘리지 않는 힘.*

## 3. 세계관 / 갈등
- **DEC** = 에너지를 무기로 쓰는 독점 세력. (현실의 자원 무기화·에너지 안보 이슈에서 모티프)
- **주인공** = 4종 공학자 중 택1 (햇살·나리·누리·새벽).
- **지역들** = 환경이 달라 각기 다른 신재생 방식에 적합 → 스테이지마다 새 에너지를 배운다.
- **여정의 형태** = 흩어진 발전을 하나씩 되살려 → 마지막에 모두 연결 → 자립 → DEC 붕괴.

---

## 4. 스테이지별 설계 (에너지 커리큘럼)

게임플레이 루프는 **모든 스테이지 공통**: 부품 3개 수집 → DEC 관문 퀴즈 통과 → 발전 복구 → 포털로 다음 지역.

### 프롤로그 — 에너지 위기 *(유지)*
- DEC의 에너지 독점·단전으로 마을이 캄캄해짐. "왜 에너지가 중요한가"를 몸으로 보여주는 도입.

### S1. 태양의 사막 — ☀ 태양광 *(현재 유지)*
- 개념: 광전효과(빛→전기), 태양광 vs 태양열, 태양 추적.
- 아이템: 태양전지 패널 / 태양추적장치 / 축전지. (현재 그대로)

### S2. 바람의 해안 — 🌬 풍력 *(현재 유지)*
- 개념: 풍력 터빈 원리, 좋은 입지(바람 센 곳), 날개 수.
- (현재 그대로)

### S3. 발명가의 도시 — ⚡ 에너지 하베스팅 *(재정렬: IP/건설/수송 → 하베스팅)*
- 컨셉: 발명가들이 **도시에 버려지는 자투리 에너지**를 줍는 장치를 만든다.
- 배울 개념: 압전(밟으면 전기 나는 바닥) · 열전(폐열→전기) · 진동/충격 수확 · "에너지 하베스팅"의 뜻.
- 수집 아이템(예): 압전 타일 / 열전 소자 / 진동 수확기.
- 퀴즈 방향(예):
  1. 사람이 많이 다니는 바닥에서 전기를 얻는 원리는? → **압전**
  2. 버려지는 열로 전기를 만드는 소자는? → **열전 소자**
  3. '에너지 하베스팅'이란? → 주변에 버려지는 작은 에너지를 모아 쓰는 것

### S4. 공장 던전 → 🔋 에너지 저장·자립 *(재정렬: 제조/재료 → 저장)*
- 컨셉: 옛 공장을 **에너지 저장 시설**로 되살린다. "낮·바람 좋을 때 모아 밤에도 쓴다."
- 배울 개념: 왜 저장이 필요한가(신재생은 들쭉날쭉) · 배터리 · 양수발전 · 수소.
- 수집 아이템(예): 대용량 배터리 / 양수발전 모형 / 수소 탱크.
- 퀴즈 방향(예):
  1. 태양광은 밤에 안 되는데 밤에도 전기를 쓰려면? → **저장(배터리)**
  2. 남는 전기로 물을 끌어올렸다가 떨어뜨려 발전하는 방식은? → **양수발전**
  3. 전기로 물을 분해해 만들어 저장·연소하는 친환경 연료는? → **수소**

### S5. 데이터 타워 — 🧠 스마트그리드 *(재정렬: 컴퓨터/AI → 에너지 관리)*
- 컨셉: DEC의 **에너지 통제 데이터타워**를 장악해, 마을들의 흩어진 신재생을 하나의 똑똑한 전력망으로 묶는다.
- 배울 개념: 스마트그리드 · 스마트미터 · 분산형 발전이 왜 안전한가(=에너지 안보) · (AI/컴퓨터가 '에너지 분배'라는 목적을 얻음).
- 수집 아이템(예): 스마트 계량기 / 그리드 제어 장치 / 연결 케이블.
- 퀴즈 방향(예):
  1. 여러 곳의 신재생 전기를 효율적으로 주고받는 똑똑한 전력망은? → **스마트그리드**
  2. 전기 사용량을 실시간으로 재고 알려주는 장치는? → **스마트미터**
  3. 한 곳이 아니라 여러 곳에서 나눠 생산하면 좋은 점은? → 한 곳이 끊겨도 안전(에너지 안보)

### S6 + 보스 — 🏛 DEC 본부: 에너지 자립 완성 *(유지, 종합)*
- 그동안 되살린 태양광·풍력·하베스팅·저장·그리드를 **모두 연결** → DEC 독점 붕괴 → 에너지 자립 선언.
- 보스전(현재 시스템 활용) → 엔딩: 마을에 빛이 돌아오고 "우리 손으로 만든 에너지" 메시지.

---

## 5. 그대로 두는 것 vs 바꾸는 것

| 그대로 (자산) | 바꾸는 것 |
|---|---|
| 엔진·이동·충돌·세이브 | S3·S4·S5의 **퀴즈·대사·아이템 이름·표지판** |
| 픽셀아트·캐릭터·오디오 | S3·S4·S5의 스테이지 **소개/연출 문구** |
| 게임 루프(수집→관문→복구→포털) | (추가) **월드맵 허브** = 에너지 여정 지도 |
| S1·S2 내용 | (추가) **홈 버튼** |
| 보스·엔딩 구조 | (다듬기) 프롤로그·엔딩의 "에너지 자립" 메시지 강화 |

## 6. 게이미피케이션 정렬 (차후, 별도 트랙)
- **에너지 도감**: 배운 개념(태양광·풍력·압전·수소·스마트그리드…)을 카드로 수집 → 학습 동기.
- **코인 상점**: 코인에 쓸 곳을 부여(꾸미기·힌트).
- **난이도**: 초/중 모드(문항 난이도 분리), 오답 복습.

## 7. 작업 순서
1. ✅ (완료) 오디오·대화창 공통화 — 기반 정비
2. **이 설계 문서 확정** ← 지금
3. 월드맵 허브 + 홈 버튼 (진행 구조)
4. S3·S4·S5 내용을 에너지로 교체 (위 4장 기준)
5. 프롤로그·엔딩 메시지 강화
6. (차후) 도감·상점·난이도 등 게이미피케이션
