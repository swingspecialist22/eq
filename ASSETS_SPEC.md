# 에너지 퀘스트 — 그래픽 에셋 제작 명세서 (v1)

> 목적: 이 문서대로 그림(에셋)을 만들면 → 제가 게임에 입혀 "확 바뀐" 비주얼을 만든다.
> **한 번에 다 안 만들어도 됨.** 우선순위 1번(배경)부터 주시면 그것부터 바로 적용합니다.

## 0. 가장 중요한 규칙 — "스타일 하나로 통일"

지금 허접해 보이는 진짜 이유는 픽셀이라서가 아니라 **다듬어진 그림이 아니라서**입니다. 아래 두 스타일 중 **하나를 골라 모든 에셋을 같은 스타일로** 만드세요. (섞이면 더 이상해 보임)

- **A. 고급 픽셀아트** — 스타듀밸리·옥토패스 느낌. 도트지만 음영·디테일이 풍부.
- **B. 깔끔한 2D(반플랫)** — 요즘 캐주얼 모바일 RPG 느낌. 부드러운 색·약간의 그림자. ← "요즘 게임처럼"에 가장 가까움(추천)

공통: **위에서 내려다보는(탑다운) 시점**, 배경 투명 PNG(배경 이미지만 예외), 밝고 선명한 색.
스테이지별 무드 색은 유지: 1 주황(사막) · 2 파랑(해안) · 3 보라(도시) · 4 주황(공장) · 5 시안(타워) · 6 마젠타(본부).

> AI(미드저니/DALL·E/SD)로 만들 땐, **모든 프롬프트에 같은 스타일 키워드를 반복**해 통일하세요.
> 예: `clean 2D top-down game art, soft shading, vibrant, mobile RPG style, ...`

## 2. 상세 스펙

### 2-1. 스테이지 배경 (먼저 이것부터!) — 6장
- **크기**: 1920 × 1080 px, PNG (불투명 OK)
- **내용**: 각 스테이지의 "먼 배경"(하늘·원경). 게임 맵이 이 위에 얹힘 → 캐릭터·UI 없이 풍경만.
- **파일명**: `bg_stage1.png` ~ `bg_stage6.png`
- **각 장면**:
  1. 태양의 사막 — 노을진 사막, 모래언덕, 멀리 태양광 발전소
  2. 바람의 해안 — 바다·해안 절벽, 멀리 풍력 터빈들
  3. 발명가의 도시 — 밝은 미래 도시 거리
  4. 공장 지대 — 거대한 산업 단지/발전소 내부
  5. 데이터 타워 — 푸른 사이버 타워 내부
  6. DEC 본부 — 어둡고 위압적인 적 요새
- **AI 프롬프트 예시**: `top-down game background, golden-hour solar desert, sand dunes, distant solar farm on horizon, [내가 고른 스타일], 16:9, no characters, no text, no UI`

### 2-2. 캐릭터 스프라이트
- **플레이어 4종**: 햇살(파란 작업복+고글), 나리(흰 연구복+안경+포니테일), 누리(주황 작업복+노란 안전모+삐죽머리), 새벽(보라 후드+마스크)
- **주요 NPC**: 촌장(흰수염), 어머니, 기술자, 발전소 박사, 각 스테이지 저항군, **DEC 적**(검은 갑옷+빨간 마크)
- **형식**: 스프라이트 시트 1캐릭터 = **4방향(아래·위·왼·오) × 3프레임(서기·걷기1·걷기2)**
  - 레이아웃: 4행(방향) × 3열(프레임), **한 칸 64×64 px** → 시트 192×256 px
  - 캐릭터는 칸 안에서 **발이 아래 중앙**에 오도록
- **배경 투명 PNG**
- **파일명**: `char_haetsal.png`, `char_nari.png`, `char_nuri.png`, `char_saebyeok.png`, `npc_elder.png`, `npc_mom.png`, `enemy_dec.png` …
- ※ AI가 시트를 깔끔히 못 뽑으면 **방향별 1장씩 따로** 줘도 됩니다(제가 시트로 조립).

### 2-3. 타일셋 (바닥·지형)
- **한 타일 64×64 px** 정사각, PNG
- **스테이지1 필요 타일**(예): 모래, 길, 오아시스(물), 선인장, 돌벽, 텐트, 태양광패널(장식), 모래언덕, 표지판, 포털, 야자수
- (스테이지마다 지형이 다르니, 1번 끝나면 2~6도 같은 식으로)
- **형식**: 개별 PNG(`tile_sand.png` …) 또는 한 장의 시트(`tiles_stage1.png`, 64 격자) — 둘 다 OK
- 바닥류는 불투명, 나무·바위 같은 장식은 투명 배경

### 2-4. 아이템 아이콘
- 수집 아이템 + 도감 카드용. **128×128 px** PNG 투명
- 목록: 태양전지 패널, 태양추적장치, 축전지 / 터빈 블레이드, 발전기, 바람지도 / 압전 타일, 단열재, 수소차 / 양수발전, 수소탱크, 배터리 / 스마트미터, 그리드모듈, AI모델
- **파일명**: `item_panel.png` `item_battery.png` … (개념 영문명)

## 3. 전달 방법
- 폴더 구조로 정리: `assets/bg/`, `assets/char/`, `assets/tiles/`, `assets/items/`
- 다 만들면 그 폴더를 **GitHub에 올리거나** 저에게 알려주세요 → 제가 렌더링 코드를 **이미지 방식으로 교체**합니다.
- **부분만 줘도 됩니다.** 예: 배경 6장만 먼저 → 그것부터 적용해 바로 차이를 봄.

## 4. 추천 진행 순서
1. **스타일 하나 결정**(A 픽셀 / B 2D — B 추천)
2. **배경 6장**부터 제작 → 전달 → 제가 적용(검은 여백이 풍경으로!)
3. 반응 보고 → 캐릭터 → 타일 → 아이템 순으로 확장
